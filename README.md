- Deployment ready for Netlify (frontend)
  
## Technologies Used

- **Frontend:** React with Vite, React-Bootstrap  
- **Version Control:** Git & GitHub  
- **Deployment:** Netlify (frontend)
- **IDE:** VS Code  

## Setup Frontend

cd frontend
npm install
npm run dev
